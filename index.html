Navegadores web

Un navegador web (en ingles, Web browser) es un software, aplicacion o programa que permite el acceso a la web, interpretando la informacion de distintos archivos y sitios web para que estos puedan ser vistos

La funcionalidad de un navegador web es permitir la visualizacion de documentos de textos, posilemente con recursos multimedia incrustados

Ejemplos de navegadores web

Safari: Safari es un navegador web de codigo desarrollado por Apple Inc. Esta disponible para MacOS, IpadOS e iOS Mozilla Firefox: Es un navegador libre de codigo abierto desrrolladopara linux, Android, iOS, macOS y Microsoft Windows coordinado por la Corpracion Mozilla y la Fundacion Mozilla. Google Chrome: Google Chrome es un navegador de codigo cerrado desarrollado por Google, aunque derivado de proyectos de codigo abierto.
